# Encryption Project

![image](https://github.com/user-attachments/assets/d89b2b13-bbe5-4c47-8d1e-92b632807ede)

## Description

Ce projet est une application Python qui permet de chiffrer et déchiffrer des fichiers texte en utilisant des techniques de cryptographie. L'objectif est de fournir une interface simple pour sécuriser des informations sensibles en les convertissant en un format illisible sans la clé appropriée.

## Contenu du dépôt

- `encryption-app.py` : Script principal contenant le code de l'application de chiffrement et de déchiffrement.

## Prérequis

- Python 3.x

## Installation

1. Clonez le dépôt :

   ```bash
   git clone https://github.com/Ivey00/EncryptionProject.git
   ```

2. Accédez au répertoire du projet :

   ```bash
   cd EncryptionProject
   ```

## Utilisation

1. Exécutez le script Python :

   ```bash
   python encryption-app.py
   ```

2. Suivez les instructions à l'écran pour choisir l'option de chiffrement ou de déchiffrement et fournissez le chemin du fichier à traiter.
